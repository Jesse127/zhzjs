## 腾讯云AI+小程序创意大赛-智慧早教树

#### 1.作品简介

智慧早教树包含国学精髓、英语学习区、闲聊机器人、图片识别四大功能。
国学精髓通过语音合成阅读内容；
英语学习区能够识别用户的发音准确度并且为用户打分；
图片识别能识别用户所上传的或拍的照片。
